⚡ Smart Routing — chọn model theo độ khó
Auto mode phân loại prompt thành 3 tier rồi chọn model phù hợp: prompt ngắn / chào hỏi → fast; prompt dài hoặc có từ khoá kỹ thuật (code, debug, viết kịch bản…) → power; còn lại → balanced. Override per-message bằng dropdown Model trong khung chat (chọn model cụ thể sẽ vô hiệu hoá auto cho lượt đó).

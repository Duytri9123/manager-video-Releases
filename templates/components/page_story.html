Nhập tên truyện rồi bấm "Tìm".
Cách dùng: Mở trang đọc chương 1 của truyện trên trình duyệt, copy URL, paste vào đây → tool tự đọc HTML và rút danh sách ảnh.
Hỗ trợ sẵn: NetTruyen, BlogTruyen, TruyenQQ, TruyenTranh, MangaBuddy. Trang khác sẽ thử fallback generic.
⚙️ Tuỳ chọn nâng cao — paste danh sách URL ảnh thủ công
Upload file ZIP chứa các trang truyện tranh. Có 2 cách OCR: tesseract (free, nhanh, kém với CJK stylised) hoặc 9Router Vision (gọi Claude/Gemini multimodal qua local 9Router → đọc cực tốt manga raw, manhua, manhwa).
9Router cần API key (lấy ở tab Chat Bot).
Dán nội dung lời thoại / lời kể của truyện. Sẽ được chia đều cho các panel ảnh.

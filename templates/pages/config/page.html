📺 Cookie YouTube
Cấu hình Cookie YouTube giúp yt-dlp tải các video bị giới hạn độ tuổi, video riêng tư, hoặc bỏ qua bot detection của YouTube.

📡 Đọc trực tiếp từ trình duyệt (Khuyên dùng)

yt-dlp sẽ tự động đọc dữ liệu cookie từ trình duyệt đã chọn. Bạn cần đăng nhập tài khoản YouTube trên trình duyệt này trước.

📄 Sử dụng File Cookie (Netscape .txt)

Dành cho VPS/Server hoặc khi không chạy trình duyệt ở môi trường local. Bạn có thể xuất file cookie bằng tiện ích mở rộng như Get cookies.txt LOCALLY.

📋 Dán nội dung Cookie (Netscape .txt hoặc chuỗi Raw)

Dán toàn bộ nội dung file cookie hoặc chuỗi cookie thô vào đây. Hệ thống sẽ tự động chuyển đổi sang file cookie định dạng chuẩn cho yt-dlp mà không cần tự quản lý đường dẫn file.

👥 Cookie & Hồ sơ Facebook
Cấu hình cookie và hồ sơ duyệt web của Facebook phục vụ cho việc cào danh sách Reels/Video và tải video chất lượng cao.

📂 Thư mục hồ sơ trình duyệt (Playwright Profile)

Đường dẫn lưu dữ liệu Chrome tự động của Playwright. Khi chạy tải Reels lần đầu, trình duyệt sẽ hiện lên để đăng nhập Facebook 1 lần duy nhất, phiên đăng nhập được lưu tại đây.

📄 Đường dẫn File Cookie Facebook (Netscape .txt)

Cung cấp file cookie dạng Netscape .txt giúp yt-dlp tải được các video Facebook chế độ HD/Riêng tư.

📋 Dán nội dung Cookie Facebook (Netscape .txt hoặc chuỗi Raw)

Hỗ trợ dán trực tiếp nội dung file Netscape hoặc chuỗi header thô. Hệ thống sẽ tự động chuyển đổi định dạng và cấu hình cho yt-dlp tải video Facebook chất lượng cao.

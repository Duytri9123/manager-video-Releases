Lấy User Access Token từ Graph API Explorer — cần permissions: pages_manage_posts, pages_read_engagement, pages_show_list.

Lưu ý quan trọng:

  • Bạn phải là admin của Page (không phải editor).
  • App của bạn phải được cấp Advanced Access cho 3 permission trên (nếu Live).
